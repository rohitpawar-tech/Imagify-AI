

# AI Image Generator UI

## Overview
AI Image Generator UI is a modern, responsive frontend web application that allows users to generate images based on text prompts. It features a futuristic interface design with smooth animations and an intuitive user experience.

This project focuses on delivering a visually appealing and interactive UI while simulating AI-based image generation workflows.

## Features

- Text-to-image prompt input
- Style selection (e.g., Realistic, Anime, Cyberpunk, Fantasy)
- Image size selection options
- Dynamic image generation (simulated API)
- Loading animation during processing
- Download generated images
- Regenerate images functionality
- Image gallery with previously generated outputs
- Fully responsive design (mobile and desktop)
- Modern UI with glassmorphism and neon effects
- Smooth animations and transitions

## How It Works

1. User enters a text prompt describing the desired image.
2. Selects preferred style and image size.
3. Clicks on the "Generate" button.
4. Application simulates an API request using JavaScript.
5. A placeholder image is generated and displayed.
6. Generated images are stored in a gallery for reuse.
